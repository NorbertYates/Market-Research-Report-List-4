<p><ul><li>Single Core Type</li><li>Two Core Type</li></ul></p>
<p><p>Phase shifting transformers are used for controlling power flow in electrical grids. Single core type transformers have one shared core for both primary and secondary windings, providing compact design and smaller footprint. Two core type transformers have separate cores for primary and secondary windings, allowing for better regulation of power flow and higher efficiency. Both types offer benefits for different applications, with single core type being more common in distribution systems and two core type being preferred for transmission systems.</p></p>
